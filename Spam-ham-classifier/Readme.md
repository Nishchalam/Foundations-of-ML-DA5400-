# Spam-Ham Email Classifier from Scratch

A complete machine learning pipeline for spam email classification implemented from scratch without relying on standard machine learning libraries.

---

## Overview

This project builds an end-to-end spam email classifier including

- Email preprocessing
- TF-IDF feature extraction
- Multiple classifiers
- Model comparison
- Evaluation pipeline

The objective is to understand classical text classification by implementing algorithms directly from their mathematical formulations.

---

## Models Implemented

- Bernoulli Naive Bayes
- Logistic Regression
- Kernel Logistic Regression
- Decision Tree
- Random Forest
- Kernel Perceptron

---

## Dataset

Training data:

- Enron Email Dataset

Evaluation:

- Automatically generated synthetic test emails
